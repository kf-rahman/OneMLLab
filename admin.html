Connect to GitHub

To publish posts, this page needs a GitHub token so it can commit files to your repo on your behalf.

  1. Go to github.com/settings/tokens/new
  2. Give it a name like OneMLLab blog
  3. Under Scopes, check repo
  4. Click Generate token and paste it below

The token is saved only in your browser's localStorage and never sent anywhere except GitHub's API.
